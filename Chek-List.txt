Проверки
1.Поля в форме регистрации и авторизации подсвечиваются зеленым и появляется галочка, если данные валидные, красным и появляется крестик, если не валидные.

2.Валидация поля "First name" и "Last name" - можно ввести латиницу, максимальное количество 50 символов для каждого поля.

3.Валидация поля “Password” - поле принимает только латиницу, обязательно должна быть одна буква в верхнем регистре, одна цифра, длина не менее 6 символов

4.После регистрации пользователь автоматически авторизуется на сайте и открывается страница личного кабинета.

5.Для авторизации на сайте пользователь нажать на кнопку в верхней правой части сайта “Регистрация”, в блоке “ALREADY REGISTERED?” ввести электронную почту и пароль.

6.Если пользователь авторизован появляется кнопка “Sign out”, и рядом отображается First name и Last name пользователя

7.В каталоге товаров есть возможность применить фильтры по: Categories, Size, Color, Properties, Compositions, Styles, Availability, Manufacturer, Condition Weight, Price.

8.Сортировать товары на странице можно по: цене (Lowest first, Highest first), по названию (Product name: A-Z, Product name: Z-A), по наличию (In stock).

9.На странице категорий товаров можно выбрать какое количество товаров показывать - 12, 24, 60 или показать все.

10.С помощью пагинации пользователь может переходить по страницам в каталоге, например с 1-ой, перейти на 2-ую, затем на 3-ю и т.д, или со 2-й перейти на 4-ю и обратно.

11.Отдельная страница товара открывается по клику на товар.

12.На странице размещаются фотографии товара, информация о товаре (название и описание), цена.

13.Поле Size - дропдаун со значение S, M, L и цвет (квадрат закрашенный определённым цветом), кнопки “Add to cart”, “Add to Wishlist”.

14.Пользователь может поделиться ссылкой на товар через социальные сети - Tweet, Facebook, Google, Pinterest.

15.По нажатию на кнопку “Add to cart” появляется модальное окно с информацией о товаре и товар добавляется в корзину.

16.В модальном окне по нажатию на кнопку “Proceed to checkout” открывается корзина с добавленными товарами.

17.Если с корзине добавлено несколько товаров, то сумма покупки суммируется

18.В корзине есть возможность удалить товар

19.Можно изменить количество выбранного товара

20.Для оформления заказа пользователь должен указать адрес доставки

21.Оплатить товар можно на сайте с помощью платежной карты
